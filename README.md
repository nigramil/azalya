# Weather Diary

**Автор:** Азалия Хусаинова

## Описание
Программа "Weather Diary" позволяет добавлять записи о погоде, фильтровать их по дате и температуре, сохранять и загружать данные из файла.

## Как запустить
1. Убедитесь, что у вас установлен Python 3.
2. Запустите файл `weather_diary.py`.
3. Введите данные о погоде и нажмите "Добавить запись".
4. Используйте поля фильтрации для поиска по дате или температуре.

## Примеры использования
- Добавление записи: дата — 2023-10-01, температура — 12, описание — солнечно, осадки — нет.
- Фильтрация по дате: введите дату и нажмите "Показать за дату".
- Фильтрация по температуре: введите минимальную температуру и нажмите "Показать выше".
